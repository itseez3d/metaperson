# LiveSpeak Samples

## Integration Sample

[LiveSpeak integration sample](https://metaperson.avatarsdk.com/livespeak/livespeak_integration_sample.html)

This sample shows how to integrate LiveSpeak into a web page via `<iframe>`, configure mode, language and optionally provide a custom avatar model.
The avatar can speak provided text or a generated chatbot response based on the given prompt.

## Microphone Sample

[Microphone sample](https://metaperson.avatarsdk.com/livespeak/microphone_sample.html)

This sample shows how to feed a raw microphone stream to LiveSpeak for animating the avatar.
