# MetaPerson Loader For Three.js

The package helps to load MetaPerson avatars in **GLB** format with the Three.js library.

The package is available in the GitHub repository: https://github.com/avatarsdk/metaperson-loader-threejs.

# Requirements

- Three.js r162 or a more recent version
- Supported platforms: Windows, Linux, MacOS, Android, iOS
