# Integration into WebGL unity applications

There are two ways of how MetaPerson Creator can be integrated into a WebGL application.

1\. Integration via an `<iframe>` element into an HTML page of the application. 

[WebGL IFrame Integration Sample](https://github.com/avatarsdk/metaperson-loader-unity/blob/main/Documentation~/MetaPersonCreatorWebGLIFrameIntegration.md)

2\. Displaying MetaPerson Creator page inside a unity scene via a WebView component.

[WebGL WebView Integration Sample](https://github.com/avatarsdk/metaperson-loader-unity/blob/main/Documentation~/MetaPersonCreatorWebGLWebViewIntegration.md)
