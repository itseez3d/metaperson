# MetaPerson Loader

MetaPerson Loader package helps to load avatars in **GLB/GLTF** format in Unity at runtime, and configures their materials and humanoid animator.

The package is available in the GitHub repository: https://github.com/avatarsdk/metaperson-loader-unity.
