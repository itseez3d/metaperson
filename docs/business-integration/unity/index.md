# MetaPerson avatars in Unity

MetaPerson avatars can be loaded into a unity scene in **GLB/GLTF** format by using our [MetaPerson Loader](metaperson_loader) package. 

If you are looking for a way to embed MetaPerson Creator page into a unity scene, please see our samples for various platforms:

 * [Windows and macOS](windows_and_macos)
 * [Android and iOS](android_and_ios)
 * [WebGL](webgl)
 * [VR](vr)
